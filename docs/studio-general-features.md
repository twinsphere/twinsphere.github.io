## ID Generator

You can specify generator patterns for several id types

- Global asset ids
- Shell ids (Twin Ids)
- Submodel ids

![twinstudio General IdGenerator](img/twinstudio-general-idgenerator.png){: width='600' }

## Language Settings

Twinstudio supports the following languages for the user interface

- German
- English

Furthermore many languages to represent the data content within twins can be specified

![twinstudio General IdGenerator](img/twinstudio-general-language.png){: width='400' }

## Tenant Switch

Twinstudio can connect to different cloud tenants depending on your permissions.
The tenants are hereby grouped into organizations. You can switch the tenant in the menu.

![twinstudio General IdGenerator](img/twinstudio-general-switchtenant.png){: width='400' }

## Dashboard

The dashboard of twinstudio serves as an entry point to the application.
You can easily switch to the twin catalog or to any other module of the application.

![twinstudio General IdGenerator](img/twinstudio-general-dashboard.png){: width='500' }
